# Tunnel Images

Place your tunnel images here. Required files:

- `tunnel-hero.jpg` — main hero image (full-bleed background). Recommended: 1920×1080px or larger.
- `hero-og.jpg` — Open Graph social share image. Recommended: 1200×630px.
- `gallery-1.jpg` through `gallery-5.jpg` — gallery images. Recommended: 1200×900px (4:3 ratio).

All images are served via Next.js `<Image>` with automatic WebP/AVIF conversion.
